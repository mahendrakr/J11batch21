METHOD:-
There are two types of method
Static method
Non static method 

Static method
you can call static method with class name.
you can also call static method with the help of object reference but it is not recommended
e.g static void m1(){}

Non static method/ Instance method
you must call instance method with the help of object reference
e.g void m1(){}
